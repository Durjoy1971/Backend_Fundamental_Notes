# Backend Fundamental Notes

A curated collection of in-depth guides covering essential backend engineering concepts — from how systems verify identity to how data moves safely between clients and servers.

Each guide is written for developers who want to understand the **why** behind the patterns, not just the **how**.

---

## What's Inside

| Guide | What You'll Learn |
|-------|-------------------|
| [Authentication & Authorization](./Authentication_and_Authorization_Guide.md) | How systems verify identity, enforce permissions, and the evolution of authentication from physical seals to modern MFA. |
| [Serialization & Deserialization](./Serialization_and_Deserialization_Guide.md) | How different languages and platforms agree on a common format to exchange structured data over a network. |
| [Validations & Transformations](./Validations_and_Transformations_Guide.md) | How to structure backend layers and validate incoming data at the entry point to prevent crashes and security issues. |

---

## Who This Is For

- Backend developers brushing up on fundamentals
- Full-stack developers transitioning deeper into server-side architecture
- Anyone preparing for system design discussions or technical interviews

---

## Prerequisites

Familiarity with basic client-server architecture, HTTP, and REST APIs is assumed. Each guide lists its own specific prerequisites at the top.
